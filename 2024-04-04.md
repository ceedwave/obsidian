## 학습내용


##### Crome 설치
```
# crome 설치 명령어
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb

# 내 디렉토리에 wget 패키지가 없어서 선행 설치 진행함
brew install wget
```

##### IP

- public 
- ping 명령어를 통해서 DNS에 연결상태 확인
- port forwording
